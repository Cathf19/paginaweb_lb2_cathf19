Bitacora - Semana 3

GitHub, Programación, CSS

¡Empecemos la semana

Comenzamos la semana repasando comandos como el tipo de letra y cómo aplicar color a los textos. Además, el profesor proporcionó orientación personalizada al resolver preguntas y dudas específicas de los estudiantes.

Posteriormente, exploramos GitHub en detalle, comprendiendo cómo esta plataforma facilita el trabajo colaborativo y el control de versiones en proyectos de desarrollo de software. Esta experiencia nos permitió ampliar nuestros conocimientos en herramientas fundamentales para la gestión eficaz de proyectos tecnológicos.

¡Sigamos aprendiendo!

En esta segunda parte, profundizamos en varios conceptos y comandos de programación relacionados con HTML. Aprendimos sobre:

  1. Text-align (Alineación de textos)
    • Left
    • Right
    • Center
    • Justify
  2. Border-width (Modificación del borde en pixles)
  3. Creación y diseño de tablas en HTML utilizando CSS para mejorar su presentación.
    • Empty-cells
    • Border-spacing
    • Border-collapse

También exploramos el CSS,que al profundizar en el tema comprendimos cómo este lenguaje de hojas de estilo complementa y mejora la estructura creada con HTML. Aprendimos a aplicar estilos, colores, márgenes, y otras propiedades visuales para dar forma y diseño a nuestros sitios web. Esta combinación de HTML y CSS nos permite no solo organizar el contenido, sino también mejorar la experiencia del usuario mediante un diseño atractivo y coherente.

Estos conocimientos prácticos nos brindaron herramientas clave para mejorar la presentación y estructura de contenido en páginas web.

RESUMEN

En esta semana empezamos recordando y aprendiendo algunos conceptos y/o comandos de programación, tambien aprendimos sobre GitHub y empezamos de poco en poco a ver lo que es CSS.
