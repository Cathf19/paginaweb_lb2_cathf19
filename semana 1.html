¡Comenzamos este viaje de nuevos aprendizajes!

Durante esta primera clase, además de conocer al instructor y recibir una visión general del curso, tuvimos la oportunidad de conectarnos como grupo a través de una actividad inicial. Esta actividad no solo evaluó nuestros conocimientos previos, sino que también facilitó la interacción entre los participantes, fomentando un ambiente colaborativo y de apoyo mutuo.

Posteriormente, nos sumergimos en las herramientas que utilizaremos en la fase inicial del curso. Exploramos su funcionalidad y aprendimos cómo sacar el máximo provecho de cada una para optimizar nuestro aprendizaje y desarrollo en el programa. Esta sesión introductoria nos preparó para abordar con confianza los desafíos y oportunidades que encontraremos en las próximas clases, estableciendo una base sólida para nuestro éxito académico y personal.

Primeros Pasos para la creación de una pagina

Después de conocernos y familiarizarnos con el curso, comenzamos aprendiendo las bases para la creación y programación de páginas web, centrándonos en HTML. HTML es un lenguaje de programacón utilizado para estructurar y desplegar contenidos en una página web.

El uso combinado de HTML y CSS nos permite separar la estructura y el contenido de la presentación, facilitando la gestión y el mantenimiento de los sitios web. Durante esta etapa inicial, exploramos conceptos fundamentales y comandos como:

  • La estructura de un documento HTML
  • !DOCTYPE html (tipo de documento)
  • Head (contenedor de metadatos no visibles para los visitantes)
  • Title (título de la página que aparece en el navegador)
  • Body (contenedor del contenido visible)
  • Los encabezados HTML (Tiene 6 niveles de encabezados → h1 - h6)
  • Las listas
    • ul (listas desordenadas)
    • ol (listas ordenadas)

Estos conceptos sientan las bases esenciales para desarrollar y diseñar páginas web funcionales y estructuradas de manera efectiva. Con este conocimiento inicial, estamos preparados para avanzar en la creación de contenido web dinámico y atractivo.
